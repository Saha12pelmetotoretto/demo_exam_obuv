# 🥿 Информационная система ООО «Обувь»

Десктопное приложение для автоматизации учёта товаров и заказов в обувном магазине. Разработано в рамках демонстрационного экзамена по специальности 09.02.07 «Информационные системы и программирование» (Модули 1 и 2, 2026 год).

## 📋 О проекте

Приложение реализует систему управления товарами, заказами и пунктами выдачи с ролевой моделью доступа. Данные импортируются из Excel-таблиц и хранятся в локальной базе данных SQLite.

### 👥 Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Гость** | Просмотр товаров (без фильтрации, сортировки и поиска) |
| **Авторизированный клиент** | Просмотр товаров (без фильтрации, сортировки и поиска) |
| **Менеджер** | Просмотр товаров с фильтрацией/поиском, просмотр заказов |
| **Администратор** | Полный CRUD для товаров и заказов, фильтрация и поиск |

## 🛠️ Технологический стек

- **Язык:** Python 3.x
- **GUI:** Tkinter
- **База данных:** SQLite3
- **Обработка данных:** Pandas, OpenPyXL
- **Версионирование:** Git, GitHub

## ⚙️ Установка и запуск

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python 3.x, затем установите необходимые библиотеки:

```bash
pip install pandas openpyxl
```
Перед первым запуском необходимо импортировать данные из Excel в SQLite:
```
python import_data.py
```
После успешного выполнения в папке появится файл database.db.
Запуск приложения
```
python main.py
```
